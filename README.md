# Blog Website (React)
## Overview
This project is a simple Blog Website built using React.
It allows users to create blog posts and view them on a main blog page. Each post includes a title, author name, and content. Clicking on a post displays the full blog content.

The project is built as part of a React assignment and demonstrates component-based architecture, routing, state management, and basic persistence using localStorage.
